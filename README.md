# Treatment_Select_viaAHP_DigitalTwin
A self-contained toolkit to keep AHP decision from real macro-economic data. Compares 3 greywater treatment technologies — membrane bioreactor (MBR), electrocoagulation (EC) and constructed wetlands (CW) — across 6 criteria (efficiency, cost, land, energy, chemical use, odor) for 3 stakeholder groups, then rolls decision forward in time
